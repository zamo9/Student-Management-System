# Student Management System

## Description
A simple web-based Student Management System built with Flask and SQLite. It allows users to:
- Add a new student
- View all students
- Update a student's details
- Delete a student

## Features
- Backend: Flask
- Database: SQLite
- Frontend: HTML, CSS, Bootstrap
